# Erhu Starter Packet

Chinese two-string spike fiddle design and build documents.

Current status: `bare-bones readiness packet`.

Fabrication authority: `not build-ready`. This repository does not yet contain
measured CAD, DXF, drawings, string data, resonator templates, membrane
selection tests, or setup measurements that can control a build.

## Packet Files

| File | Purpose |
| --- | --- |
| `design.md` | Design intent, governing measurements, assumptions, and open questions. |
| `family-spec.csv` | Starter family/spec record for strings, qianjin, bridge, resonator, and membrane. |
| `bom.csv` | Starter bill of materials with verification status. |
| `sourcing.csv` | Sourcing and provenance placeholders without current purchase claims. |
| `fabrication-notes.md` | Early fabrication sequence and setup notes for a future measured packet. |
| `validation-loop.csv` | Measurement loop for pitch, setup, bow response, membrane behavior, and updates. |
| `visual-output-register.csv` | Visual/CAD authority register showing current placeholders only. |
| `wolfram-study-notes.md` | Optional unvalidated study notes for string tension and resonator exploration. |

## Next Gates
